# Tabela de produtos com React + Typescript

Mini-projeto com o propósito de aumentar a compreenção do Typescript, implementando tipagem as funcionalidades do código.

> API utilizada: [Ranek Origamid](https://ranekapi.origamid.dev/json/api/produto)
